{% alert important %}
Braze は、500万 を超えるセッションでユーザーを禁止またはブロックします("ダミーのusers")。通常、それらは誤統合の結果であるため、SDK イベントを取り込むことはありません。正規ユーザーに対して禁止またはブロックが発生しているとわかった場合は、Braze アカウントマネージャーに連絡してください。
{% endalert %}
